外部组件（5）

写完浮层入口的样式，接下来将写header的公告部分。
公告部分也是点击 > 则会弹出浮层。
公告的文字超出可显示的范围则用...代替，设置
white-space: nowrap;
overflow: hidden;
text-overflow: ellipsis;
如果父级元素设置font-size:0;则...会消失，
所以为了消除行内、行内块级元素之间的空白间隔，要使用别的方法：
解决方法:
1.	给父元素设置font-size:0px
2.	取消掉换行符，如这样：<span>aaaa</span><span>aaaa</span><span>aaaa</span>连续。
3.	使用margin负值。
4.	使用浮动。（块元素同行显示：①display：inline-block②float：left）
5.	另外还有使用letter-spacing、word-spacing等方法。


.bulletin-wrapper{
    height: 28px;
    line-height: 28px;
    padding: 0 22px 0 12px;
    white-space: nowrap;
    overflow: hidden;
    text-overflow: ellipsis;
    .bulletin-title{
      display: inline-block;
      vertical-align: top;
      margin-top: 10px;
      width: 22px;
      height: 12px;
      background: url(bulletin@2x.1.png) no-repeat;
      background-size: cover;
    }
    .bulletin-text{
      font-size: 10px;
      margin: 0 4px;
    }
}

.icon-keyboard_arrow_right使用绝对定位。不占据文档流。设置样式：
.icon-keyboard_arrow_right{
        position: absolute;
        right: 12px;
        top: 8px;
        font-size: 10px;
        line-height: 15px;
}

接下来给公告栏设置背景：
.bulletin-wrapper{
    background: rgba(7,17,27,0.2);
}
